# MongoDB Lecture Notes

This repository contains lecture materials and practice problems for MongoDB, specifically covering Schema Design and Practical Applications.

## Contents

- **Chapter 13: MongoDB Schema**
  - Concepts of MongoDB Schema (Schema-less vs. Structured)
  - MongoDB Design Patterns
  - Data Modeling (One-to-One, One-to-Many, Many-to-Many)
  - Performance and Scalability considerations (Sharding, Partitioning)
  
- **Chapter 14: MongoDB Practice**
  - Designing a Content Platform (Blog/Post/Comment system)
  - Relationship modeling (Embedding vs. Referencing)
  - Practical examples of Post, User, Comment, Category, and Tag models
  - Implementation examples using Mongoose-like syntax

## Files

- `13. MongoDB Schema.pdf / .txt`: Lecture notes on schema design.
- `14. MongoDB Practice.pdf / .txt`: Practical requirements and modeling examples.
- `mongodb_lecture.html`: Combined lecture material in HTML format.
- `template.html`: Base template for the lecture materials.
